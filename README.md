# 📚 Kurshanterare (TypeScript + Vite)

En enkel webbapplikation för att hantera kurser.  
Applikationen är byggd med **TypeScript** och använder **localStorage** för att spara data mellan sessioner.




👉 Länk till publicerad webbplats: https://coco10023.github.io/kursapp/ 


## 📦 Funktioner

- ➕ Lägg till nya kurser
- 📄 Visa kurslista
- ❌ Radera kurser
- 💾 Spara kurser i localStorage
- ✅ Validering av:
  - Unik kurskod
  - Progression (A, B eller C)
  - Tomma fält
